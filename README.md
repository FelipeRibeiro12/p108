# Teoria das Filas - Simulador

Este projeto é um simulador visual de modelos clássicos de filas (M/M/1, M/M/c, M/G/1, com e sem prioridade, população finita, capacidade limitada, etc.), desenvolvido em Python com interface gráfica usando Pygame.

## Funcionalidades

- Simulação de diversos modelos de filas;
- Interface gráfica interativa para entrada de parâmetros e visualização dos resultados;
- Cálculo automático de métricas como tempo médio, número médio de clientes, probabilidades, custos, etc.

## Instalação

1. Certifique-se de ter o Python 3 instalado.
2. Instale as dependências do projeto:

```cmd
pip install -r requirements.txt
```

## Como executar

No terminal, execute:

```cmd
python main.py
```
